# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
# Build
go build -o inkfeed-backend

# Run (defaults to port 8080)
./inkfeed-backend
./inkfeed-backend -port 9090

# Run directly
go run .

# Test
go test ./...

# Dependencies
go mod tidy

# Regenerate sqlc queries (after editing db/query.sql)
sqlc generate
```

## Environment Variables

Copy `.env` and configure:

| Variable | Default | Description |
|---|---|---|
| `PORT` | `8080` | Server port |
| `ENV` | — | Set to `local` to allow `http://localhost:8000` CORS origin |
| `EMAIL_PROVIDER` | — | `brevo`, `mailersend`, or `resend` |
| `EMAIL_API_KEY` | — | API key for chosen email provider |
| `EMAIL_FROM` / `EMAIL_FROM_NAME` | — | Sender identity |
| `RATE_LIMIT_MAX` | `40` | Requests per window |
| `RATE_LIMIT_WINDOW_SECONDS` | `60` | Rate limit window |
| `IMAGE_QUALITY` | `50` | JPEG compression quality (1–100) |
| `IMAGE_COMPRESSION` | `true` | Set `false` to disable |
| `FEED_PROXY_URL` | (Cloudflare Worker URL) | Proxy for fetching feeds |
| `ARTICLE_CACHE_TTL_MINUTES` | `60` | Article persistent cache TTL |
| `COMMENTS_CACHE_TTL_MINUTES` | `30` | Comments persistent cache TTL |
| `FEED_SCRAPE_INTERVAL_HOURS` | `1` | How often to scrape saved feeds |
| `CONTENT_ARCHIVER_TIMEOUT_SECONDS` | `5` | HTTP timeout for background article archiving |

## Architecture

Single-package Go application. All source files live in the root directory. Uses a SQLite database (`inkfeed.db`) for user accounts, sessions, preferences, saved feeds, feed groups, favorites, and persistent cache.

### Database

- SQLite via `modernc.org/sqlite` (pure Go, no cgo)
- Schema is auto-migrated on startup in `main.go`
- Queries are generated by **sqlc** — edit `db/query.sql` and run `sqlc generate` to update `db/query.sql.go`; do not edit generated files manually
- `db/schema.sql` is the canonical schema definition

### Middleware pipeline (applied per route in `main.go`)

```
loggingMiddleware → corsMiddleware → authMiddleware → rateLimitMiddleware → cached → Handler
```

- **`auth.go`** — Session-based auth using `bcrypt` passwords and random hex session tokens stored in SQLite. Sessions last 15 days, delivered via `HttpOnly` cookie. `authMiddleware` validates the cookie and injects `userID` into request context.
- **`ratelimit.go`** — In-memory per-IP rate limiting; reads `True-Client-IP` header (Render proxy). Separate stricter rate limiters for `/signup`, `/signin`, and `/email`.
- **`cache.go`** — Two caching layers:
  - In-memory 5-minute cache (feed responses), keyed by full URL
  - Persistent SQLite cache (`persistentCached`) used for article/comments with configurable TTL

### Handlers

| File | Routes | Purpose |
|---|---|---|
| `auth.go` | `POST /signup`, `POST /signin`, `POST /signout` | User registration and session management |
| `preferences.go` | `GET,PUT /preferences`, `PUT /saved-feeds`, `PUT /feed-groups`, `PUT /favorites` | Per-user preferences, saved feeds, feed groups, and favorites stored in SQLite |
| `feed.go` | `GET /feed` | RSS/Atom parsing via gofeed; appends `/.json` for Reddit items |
| `article.go` | `GET /article`, `GET /text` | Readability-based article extraction and plain-text export |
| `comments.go` | `GET /comments` | Reddit (native JSON) and HN (Algolia API) comment rendering; falls back to Readability |
| `epub.go` | `POST /epub` | EPUB generation via go-shiori/go-epub; handles bulk with concurrency (max 5 goroutines) |
| `download.go` | `POST /mobi` | MOBI generation delegating to `mobi/writer.go` |
| `email.go` | `POST /email`, `POST /email-file` | Attaches EPUB/MOBI to transactional email via provider adapter |
| `reddit.go` | `GET /reddit-post` | Reddit self-post HTML extraction |
| `googlenews.go` | `GET /decode-google-news` | Decodes obfuscated Google News redirect URLs |

### Email provider adapter pattern

`email.go` defines an `EmailSender` interface. Provider implementations (Brevo, MailerSend, Resend) are selected at runtime via `EMAIL_PROVIDER`. To add a new provider, implement the interface and add a case in the factory function.

### MOBI format (`mobi/writer.go`)

Custom Go port of a Mobipocket writer. Chunks HTML into 4096-byte records. Used for Kindle compatibility where EPUB is insufficient.

### Image handling in EPUB

WebP images are converted to JPEG (Amazon Kindle doesn't support WebP). Compression is applied based on `IMAGE_QUALITY`. Images are embedded directly in the EPUB archive.

### CORS

Restricted to `https://reader.inkfeed.xyz` in production. Set `ENV=local` to allow `http://localhost:8000` instead.
